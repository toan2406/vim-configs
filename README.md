# Usage

#### Refresh
* Edit file `:e`
* Refresh NERDTree `R`

#### Comment
* Comment lines `;c<space>` `;cm`
* Comment selected `;cc`
* Uncomment `;cu`

#### Select
* Visual a block `va{`
* Visual inside block `vi{`
* Visual a word `vaw`
* Visual inside word `viw`
* Visual 3 words `v3aw`
* Visual a paragraph `vap`

#### Surround
* Surround selected `S"`
* Surround inside word `ysiw"`

#### Navigate
* Jump to next char `f<char>`

#### Find
* Find word at cursor `*`
